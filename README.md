# 🛠️ Projects Developers — Deep AI Gateway
### *The Advanced Engineering Portal for Projects Bots*

بوابة المطورين هي المنصة التقنية المخصصة لفحص، تحليل، واعتماد المشاريع البرمجية قبل دمجها في الموقع الأساسي **Projects Bots**. تعتمد البوابة على محرك ذكاء اصطناعي عميق لضمان جودة الكود وخلوه من الثغرات الأمنية.

---

## 🚀 نظرة عامة (Overview)
هذا المستودع (Repository) يمثل المحطة الأولى لأي مطور يرغب في نشر عمله. النظام هنا ليس مجرد واجهة رفع ملفات، بل هو "بيئة اختبار" (Sandbox) متكاملة تقوم بـ:
1. **التحقق من الهوية:** قبول المطورين بناءً على تحليل AI لمهاراتهم.
2. **الفحص العميق (Deep Scan):** تحليل سورس كود المشاريع (JS/CSS) واكتشاف الأخطاء المنطقية والأمنية.
3. **المزامنة الذكية:** بعد الموافقة، يتم توجيه المطور تلقائياً للموقع الرئيسي لبدء النشر الرسمي.

---

## 🧠 تقنيات الفحص (Scanning Core)
يستخدم النظام موديل `claude-sonnet-4-20250514` لإجراء عمليات:
- **Static Analysis:** تحليل الكود بدون تشغيله لاكتشاف الثغرات.
- **Logic Validation:** التأكد من أن منطق البرمجية سليم ولا يستهلك موارد بشكل عشوائي.
- **Security Audit:** فحص الوصلات الخارجية والحماية من حقن الكود الخبيث.

---

## 📂 هيكلية المشروع (Project Structure)
بما أننا نعتمد مبدأ الفولدر المفتوح (No Folder Strategy)، يتكون المشروع من:
- `index.html`: الواجهة الرئيسية التي تجمع بين نظام التقديم ولوحة التحكم.
- `dev-style.css`: التصميم المستقبلي المعتمد على الـ Dark Mode والفيكتورات.
- `dev-script.js`: المحرك البرمجي الذي يدير عمليات الـ API والـ Deep Scan.

## 🛠️ تعليمات التشغيل (Setup)
1. قم برفع الملفات الثلاثة على استضافة **GitHub Pages**.
2. تأكد من تفعيل مفتاح الـ API الخاص بـ Anthropic في ملف `dev-script.js` (لأغراض التطوير).
3. للوصول إلى لوحة المطورين مباشرة، يجب اجتياز اختبار الـ AI في صفحة الانضمام.
